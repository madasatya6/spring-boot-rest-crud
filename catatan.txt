1. masuk ke command pallete: >spring 
2. pilih "Spring Initializr: Create a Maven Project"
3. pilih versi spring bebas
4. pilih Bahasa pakai java
5. masukkan nama package
6. masukan nama aplikasi
7. pilih JAR
8. pilih versi java
